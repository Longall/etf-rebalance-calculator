# ETF Rebalance iOS PWA Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将现有静态计算器变成可安装到 iPhone 主屏幕并离线启动的 `ETF Rebalance` PWA。

**Architecture:** 使用 Manifest 描述安装元数据，版本化 Service Worker 只缓存同源应用外壳，独立 `pwa.js` 负责注册、在线状态和用户确认更新。行情请求保持网络直连，不进入缓存。

**Tech Stack:** 原生 HTML/CSS/JavaScript、Web App Manifest、Service Worker、localStorage、Node.js `node:test`。
